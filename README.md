# 🎮 UP - Platform Jump Game

A simple and fun **2D platform jumping game** made using Python and Pygame.  
Jump between platforms, avoid falling, and try to beat your high score!

---

## 🚀 Features

- 🕹️ Smooth player movement (Left / Right)
- ⬆️ Jump using **SPACE**
- 🧠 Platform generation system (infinite gameplay)
- 🎥 Camera scrolling
- 🧾 Score + High Score system
- 🎵 Sound effects (jump, game over, buttons)
- 🧩 Main menu with buttons:
  - Play
  - Restart
  - Exit
  - Back to Menu

---

## 🎮 Controls

| Key        | Action              |
|------------|--------------------|
| ⬅️ Left     | Move Left          |
| ➡️ Right    | Move Right         |
| Space      | Jump               |
| Esc        | (Future Menu)      |
